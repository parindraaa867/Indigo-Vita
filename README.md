# Indigo-Vita
Touge racer with inspiration from Initial D, developed in Unity. 

Overclock required,
because this is a Unity-based Vita game with custom physics, it will not run well on stock hardware speeds. You must use an overclocking plugin (like PSVshell) and set your clocks to the maximum


If you try to run this without these exact settings, the framerate will tank and the game will be unplayable.
